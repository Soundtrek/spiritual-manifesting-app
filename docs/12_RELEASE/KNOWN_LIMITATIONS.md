# Known Limitations

## Current Limitations

- Data is saved in the current browser only through local storage.
- Data does not sync across devices, browsers, or accounts.
- Authentication and account recovery are not active.
- AI reflection is not active yet; reflection surfaces are static or user-entered only.
- No backend database storage is implemented.
- No export or import workflow exists yet.
- Clearing local data affects only this browser's stored SMA data.
- Production deployment is not guaranteed by the current local MVP foundation.
