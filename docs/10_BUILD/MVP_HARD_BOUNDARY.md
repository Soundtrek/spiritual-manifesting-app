# MVP Hard Boundary

Canonical path: `docs/10_BUILD/MVP_HARD_BOUNDARY.md`

## MVP Includes

- Daily intention.
- Affirmation generator placeholder or later grounded generator.
- Gratitude journal.
- Manifestation board.
- Mood and energy check-in.
- Reflection history.
- Simple AI guide when explicitly scoped in a later module.
- Weekly insight summary when explicitly scoped in a later module.

## Not In MVP Yet

- Payments.
- Social features.
- Voice AI.
- Tarot.
- Dream interpretation.
- Wearable integration.

## Build Guardrails

- Do not add AI, auth, persistence, database, Supabase, payments, notifications, or backend routes unless the active module explicitly asks for them.
- Prefer static or local-state implementation for foundation modules.
- Keep spiritual language grounded and non-promissory.
- Preserve user agency and avoid dependency-building behavior.
