# MVP Scope

## MVP Goal

Deliver a calm AI-assisted daily reflection and manifestation app with journaling, intentions, affirmations, and weekly reflection.

## MVP Features

- Daily alignment flow
- Mood check-in
- Gratitude entry
- AI affirmations
- Manifestation board
- Journal history
- Weekly AI reflection
- Basic account system

## Excluded From MVP

- Social features
- Public profiles
- Marketplace
- Voice AI
- Tarot systems
- Wearables
- Complex agents
