# Project Brief

## Purpose

A calm, grounded spiritual manifesting app for intention setting, journaling, affirmations, visualization, rituals, and AI-supported reflection.

## Core Promise

Help users reflect, focus, and build gentle daily practices around their intentions.

## Core Boundary

The app must not promise supernatural, financial, medical, relationship, or life outcomes.

## Current Phase

Planning-first foundation. No production build until the core scope, safety rules, user journeys, and data model are stable.
