# Non-Goals

This document protects the project from scope creep.

## Not Building Now

- Social network features
- Public manifestation feeds
- Marketplace features
- Payments in MVP
- Tarot prediction as truth
- Medical, mental health, legal, or financial advice
- Voice companions
- Wearable integrations
- Over-complex AI agent systems

## Always Forbidden

- Guaranteed outcomes
- Emotional manipulation
- Dependency-building language
- Claims that the AI knows the user's destiny
