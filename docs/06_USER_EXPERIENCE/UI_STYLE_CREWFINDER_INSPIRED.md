# UI Style Crewfinder Inspired

Canonical path: `docs/06_USER_EXPERIENCE/UI_STYLE_CREWFINDER_INSPIRED.md`

## Direction

Use a calm, modular, dark-mode-first interface inspired by focused operational tools: clear surfaces, simple hierarchy, compact content blocks, and predictable navigation.

## Principles

- Mobile-first layouts with clear spacing and readable type.
- Dark surfaces with restrained contrast.
- Cards for discrete repeated content or form sections, not decorative nesting.
- Plain, grounded copy over mystical overclaiming.
- Strong action clarity: users should understand what each screen lets them do.

## Avoid

- Marketing-style landing pages for app workflows.
- Overly decorative gradients or ornamental visuals.
- Claims that imply guaranteed manifestation outcomes.
- UI text that creates urgency, pressure, or dependency.
