# Governance README

## Purpose

The governance folder defines the project rules that protect the app from becoming unsafe, manipulative, over-scoped, or incoherent.

## Documents

- AI_SAFETY_RULES.md
- ETHICAL_BOUNDARIES.md
- HUMAN_AUTHORITY_MODEL.md
- CONTENT_POLICY.md
- PRIVACY_PRINCIPLES.md
- DECISION_LOG.md
- ARCHITECTURAL_OWNERSHIP_MAP.md

## Rule for Future Work

Before adding features, prompts, AI behaviors, monetization, or user-data handling, check this folder first.

## SAMMA Requirement

This folder acts as the early governance layer for AI-assisted development. It gives future AI/coding tools a stable set of boundaries before implementation begins.
