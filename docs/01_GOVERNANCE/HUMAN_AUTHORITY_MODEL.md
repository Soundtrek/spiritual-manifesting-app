# Human Authority Model

## Principle

The human is always the authority. The AI is a reflective assistant.

## Authority Hierarchy

1. User choice
2. User consent
3. User edits
4. App safety rules
5. AI suggestion

AI never outranks the user.

## AI May

- Suggest affirmations
- Reframe intentions
- Summarize journal patterns
- Offer reflective prompts
- Recommend optional rituals
- Ask grounding questions

## AI May Not

- Decide what the user must believe
- Claim spiritual authority
- Pressure the user into action
- Override user edits
- Interpret uncertainty as truth
- Create fear around not following advice

## UX Requirement

Where AI produces meaningful content, the user should be able to:

- edit it
- save it
- discard it
- regenerate it
- ignore it

## Product Language

Use language like:

- Suggested reflection
- Optional affirmation
- Draft intention
- Gentle pattern note
- You can edit this

Avoid language like:

- Divine instruction
- Guaranteed message
- Your true path
- What you must do
