# Local QA Checklist

## Scope

Manual QA for the local browser-only MVP.

## Checklist

- Open `/` and confirm dashboard navigation works.
- Complete `/daily`, reload, and confirm saved fields remain.
- Add, edit, update status, delete, and reload `/intentions`.
- Add, edit, filter, mark gratitude, delete, and reload `/journal`.
- Open `/reflection` and confirm dashboard panels render without fake AI output.
- Open `/settings` and confirm Local MVP status is accurate.
- Use Settings clear-local-data, confirm, and verify Daily, Intentions, and Journal reset in this browser.
- Resize to mobile width and confirm pages remain readable and usable.
- Review visible copy for grounded language, user agency, and no guaranteed outcomes.
- Review docs/12_RELEASE/KNOWN_LIMITATIONS.md before beta handoff.

## Pass Standard

All checklist items should pass locally before inviting beta feedback. Any failed item should be logged with page, browser, steps, expected result, and actual result.
